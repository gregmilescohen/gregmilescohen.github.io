Ruby on Rails

What is Rails?

3.5.15

So far in DBC Phase 0, we have only been raw coding in a text editor. We have not used any type of framework. Because I started coding the same time I started DBC, this is all I know. From what I understand, in the future I will always be working in a framework, and more specifically using the framework Ruby on Rails. But I want to delve into what Rails actually is so that I have a better understanding of what I’m getting myself in to.

Wikipedia tells us “Rails is a full-stack framework that emphasizes the use of well-known software engineering patterns and paradigms, including convention over configuration (CoC), don't repeat yourself (DRY), the active record pattern, and model–view–controller (MVC).” Considering my coding experience is a mere 8 weeks old, this technical description does not help me much, and probably won’t help any other beginners either.

Ruby on Rails (usually just referred to as “Rails”) is a software library that adds to and expands upon the Ruby language. But why would we want to use Rails? Well Rails is a framework that makes building a website much easier. And we all want easier. The framework combines Ruby, HTML, CSS, and JavaScript so that we can easily build web applications and complex web sites. The main difference between “coding raw” (what I’m doing now using Sublime as my text editor) and using the Rails framework, is that right now outside of a framework I can find one solution a hundred different ways. Whereas with Rails, there are often times one particular way to do something. I believe that I prefer the latter. I like knowing that there is one best practice to perform a task. It avoids confusion.

What I believe may be more important than Ruby on Rails itself is the community that surrounds it. Rails has become so popular in large part because of the massive amount of open-source software available to it. This has attracted businesses, investors, and recruiters alike to the Rails community. I think that just being part of the Rails community will help anyone in their career, which is what makes me so excited.
