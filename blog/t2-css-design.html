CSS: Classes & IDs

What's The Difference?

1.24.2015

I started learning CSS just three days ago, and I cannot believe how much I have already been able to accomplish. One thing I’ve learned that is very important is the usage of “classes” and “IDs.” These properties allow us to adjust the parameters of a single element or multiple elements.

Classes are great for when you have multiple elements that need the same styling. The CSS syntax for a class looks like this:

.object {

height: 2em;

width: 2em;

}

IDs are great for when you have a single element that needs to be styled a certain way. This allows for some serious customization! The CSS syntax for an id looks like this:

#text {

color: #000000;

font-family: sans-serif;

}

Classes and IDs have certainly helped me, so I hope this information helps some of you as well!
